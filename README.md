# Firebase Notification App

This Flutter app is a practical starter project for integrating Firebase and notification features into a mobile application. It demonstrates Firebase initialization, local scheduled notifications, and a simple UI for triggering test notifications.

## Features

- Firebase integration with Flutter
- Push notification handling for app-based notification workflows
- Local and scheduled notification demo
- Simple home screen for testing notifications
- Android/iOS-ready project structure
- Firebase configuration support via FlutterFire

## Tech Stack

- Flutter
- Dart
- Firebase Core
- Firebase Messaging
- Flutter Local Notifications
- Timezone support
- Permission handling

## Project Structure

## Prerequisites

- Flutter SDK installed
- Android Studio or Xcode configured for mobile development
- A Firebase project created in the Firebase Console

## Setup Instructions

1. Clone the repository
   ```bash
   git clone https://github.com/fkirewold/firebase-notification-app.git
   cd firebase-notification-app
   ```

2. Install dependencies
   ```bash
   flutter pub get
   ```

3. Configure Firebase
   - Create a Firebase project in the Firebase Console
   - Register your Android and/or iOS app
   - Download the required configuration files:
     - Android: google-services.json
     - iOS: GoogleService-Info.plist
   - Place them in the appropriate platform folders
   - Run the following command to generate Firebase options:
     ```bash
     flutterfire configure
     ```

4. Run the app
   ```bash
   flutter run
   ```

## Usage

Open the app and tap the button on the main screen to trigger a test notification after 3 seconds.
